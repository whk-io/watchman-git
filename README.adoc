# Watchman-Git 

Makefile and trigger script for rapidly setting up auto git commits on changes with watchman.

TIP: Requires Make, Watchman, Git, and Docker if you want to build the container.

* `make` to view the available options
* `make list` : List watchman watched folders
* `make list-triggers` : list triggers for current folder
* `make start` : Start watching the current folder
* `make stop` : Stops watching the current folder
* `make trigger` : Create trigger for current folder
* `make state` : View watchman state
* `make log` : View watchman log
* `make monitor` : Stream active log
* `make build` : Build watchman-git container
* `make debug` : Run container for debugging
* `make run` : Run container and start a watch and trigger.
