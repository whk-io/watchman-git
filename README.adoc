# Watchman-Git 

Makefile and trigger script for easily setting up auto commits on changes with watchman.

TIP: Requires Make, Watchman, Git

* `make` to view the available options
* `make list` : List watchman watched folders
* `make list-triggers` : list triggers for current folder
